# Vun Tình

Vun Tình
Đơm vun vun sôi
đĩa đầy
Nâng hoa huệ
trắng
Lọ đồng cổ cao
Thắp hương
rủ Nguyệt-Lão
về
thắt nơ tình-ái
buộc Đời-nhau
tươi
Tiết Đông
gió đón
Tết sang
Góc riêng
chung
hội vui
quanh địa-cầu
