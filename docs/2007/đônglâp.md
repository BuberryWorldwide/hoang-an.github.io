# đông-Lập

Hoàng-Ân

ĐÔNG-LẬP
trắng cây cành vương hơi sương mù
giá buốt tay đón gió đầu Đông
chim hót rỡ-ràng xù lông trơ mỏ khuyên
trắng cây cành vương mù hơi sương
