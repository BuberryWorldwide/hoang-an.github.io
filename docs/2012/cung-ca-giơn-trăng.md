# Cùng Cá Giởn Trăng

Hoàng-Ân


CÙNG CÁ GIỞN TRĂNG

vẩy cá bạc lóng-lánh
vờn ánh ngọt sóng thầm-thì
trăng tròn soi vằng-vặc
tung-tăng cá quây¬ trăng

tôi lặn trầm dưới sóng
thăm loài cá giơn¬ chơi

và ướt dượt cát bờ
tôi dâng giọng ca cho
phù-thủy tình yêu

...hôn trăng tìm dáng
   Chàng..!!
